**java对象计算方式**

Java  基本对象占用内存大小

| 数据类型 | 所占空间（byte） |
| :------- | ---------------- |
| byte     | 1                |
| short    | 2                |
| int      | 4                |
| long     | 8                |
| float    | 4                |
| double   | 8                |
| char     | 2                |
| boolean  | 1                |
